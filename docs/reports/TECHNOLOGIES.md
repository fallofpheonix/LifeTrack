# Technologies

## Languages
- Dart (SDK ^3.10.8)

## Framework
- Flutter

## State Management
- `flutter_riverpod`: Primary state management.
- `provider`: Legacy support.

## Local Storage and Security
- `drift`: SQLite persistence.
- `sqlite3_flutter_libs`: Native SQLite support.
- `shared_preferences`: Simple key-value storage.
- `flutter_secure_storage`: Secure key-value storage.
- `encrypt`: Data encryption.
- `path_provider`: Filesystem access.

## Domain/Utilities
- `intl`: Internationalization and date formatting.
- `uuid`: Unique ID generation.
- `http`: Network access.
- `xml`: XML parsing.
- `url_launcher`: Opening external links.
- `share_plus`: Content sharing.
- `package_info_plus`: Versioning metadata.

## Sensors and Visualization
- `pedometer`: Step counting.
- `fl_chart`: Health data visualization.

## Tooling
- `drift_dev`: Code generation for Drift.
- `build_runner`: General code generation.

## Testing
- `flutter_test`: Unit and widget testing.
